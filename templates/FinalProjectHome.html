Research Objective

This project investigates whether human thinking time in chess can be predicted from observable features of the game state. Focusing on blitz games played by Magnus Carlsen, we aim to model the time spent on each move as a function of temporal, positional, and game-dynamic variables.

Scientific Context

Modern chess engines vastly outperform humans in move quality, yet fail to reproduce an essential aspect of human cognition: time management. Human players hesitate, rush, and adapt their thinking time to context. This temporal dimension is largely absent from current AI systems.

By modeling thinking time, this project contributes to the broader goal of human-aligned chess AI, complementing existing imitation-based approaches such as Maia.

Dataset

The dataset is built from 250 blitz games played by Magnus Carlsen on chess.com. Each move is extracted from PGN files and enriched using python-chess.

  • One row per move played by Magnus Carlsen
  • Temporal features (clock, increment, previous move time)
  • Positional and game-dynamic features

Large files and processed datasets are archived on Zenodo.
