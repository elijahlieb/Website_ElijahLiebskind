Objective

The goal of this project is to build a machine learning model capable of predicting Magnus Carlsen's thinking time for a given move.

Context

Since the early 2000s, computers have been outperforming the best chess players. However, reproducing human time management is an interesting challenge for data science.

Methodology

  1. Data preprocessing and variable creation
  2. Exploratory analysis
  3. Training the model
  4. Evaluating its performance
